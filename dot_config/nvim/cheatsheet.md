# Raccourcis personnalisés | custom

## Fichiers | files
    <leader>ff    Fuzzy find fichiers
    <leader>fg    Grep dans le projet
    <leader>e     File explorer (neo-tree)
    <C-p>         Find files (telescope)

## Git | git
    <leader>gg    Lazygit (float)
    <leader>gd    Lazydocker (float)
    <leader>gb    Git blame ligne

## Sessions | sessions
    <C-f>         Sessionizer (switch projet)
    <leader>?     Ce cheatsheet

## Tmux | tmux
    C-h/j/k/l     Navigate panes (tmux + neovim)
    <prefix>|     Split vertical
    <prefix>-     Split horizontal
    <prefix>w     Overview toutes les fenetres

## Workmux | workmux
    workmux add <feature>     Nouveau worktree + window tmux
    workmux merge             Merge + cleanup
    workmux list              Lister les worktrees actifs

## Divers | misc
    <C-s>         Sauvegarder
    <S-h>/<S-l>   Buffer precedent/suivant
    <leader>gg    Lazygit
